# Network Packet Sniffer

## Description
A Python-based tool designed to capture and analyze network packets. This project helps in understanding how data travels across a network and identifying potential vulnerabilities.

## Key Features
- Real-time packet capturing.
- Filters traffic based on protocols.
- Extracts source and destination IP addresses.

## How to Run
1. Ensure you have Python installed.
2. Install the necessary library: `pip install scapy`
3. Execute the script: `python3 network_sniffer.py`

## Technologies
- Python, Scapy Library
