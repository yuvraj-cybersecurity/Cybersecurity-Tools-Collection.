# Activity Logger

## Description
A Python-based utility that logs keystrokes and system activity. This project is designed for educational purposes to understand how input monitoring works in security auditing.

## Features
- Records keystrokes to a local file.
- Lightweight and simple implementation.
- Useful for understanding system event monitoring.

## How to Run
1. Ensure Python is installed.
2. Install the necessary library: `pip install pynput`
3. Run the script: `python3 activity_logger.py`
