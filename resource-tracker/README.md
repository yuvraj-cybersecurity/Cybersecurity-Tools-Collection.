# Resource Tracker

## Description
A Python script to monitor system resources like CPU usage and RAM. This is useful for identifying performance bottlenecks or potential malware that may be consuming excessive system resources.

## Features
- Real-time CPU usage monitoring.
- Memory/RAM usage statistics.
- Simple, text-based dashboard.

## How to Run
1. Ensure Python is installed.
2. Install dependencies: `pip install psutil`
3. Run the script: `python3 resource_tracker.py`
