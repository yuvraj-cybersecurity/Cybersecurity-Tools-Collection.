# Net Check

## Description
A Python-based utility to monitor network connectivity and verify host reachability. It provides quick feedback on whether a network interface or a specific host is online.

## Features
- Real-time connectivity status.
- Verifies host reachability via Ping.
- Simple and lightweight design.

## How to Run
1. Ensure Python is installed.
2. Run the script: `python3 net_check.py`
