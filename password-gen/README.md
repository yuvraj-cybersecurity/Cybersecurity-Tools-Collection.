# Password Generator

## Description
A secure and efficient Python tool that generates strong, random passwords to enhance digital security. This script allows users to specify password length and character complexity.

## Features
- Customizable password length.
- Option to include uppercase, lowercase, numbers, and symbols.
- High entropy for better security.

## How to Run
1. Ensure Python is installed.
2. Run the script: `python3 password_gen.py`
