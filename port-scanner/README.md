# Port Scanner

## Description
A fast and efficient Python-based port scanner used to identify open ports on a target host. This tool is essential for network auditing and identifying potential entry points in a network.

## Features
- Scans a range of ports on a specified IP address.
- Identifies open, closed, or filtered ports.
- Provides real-time status updates.

## How to Run
1. Ensure Python is installed.
2. Run the script: `python3 port_scanner.py`
3. Enter the target IP when prompted.
