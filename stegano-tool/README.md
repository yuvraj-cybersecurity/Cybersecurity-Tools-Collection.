# Stegano Tool

## Description
A Python-based steganography tool that allows users to hide secret messages or data within image files. This project explores the fundamentals of data obfuscation and digital security.

## Features
- Embed data into image files.
- Extract hidden data from images.
- Supports common image formats.

## How to Run
1. Ensure Python is installed.
2. Install dependencies: `pip install stegano`
3. Run the script: `python3 stegano_tool.py`
