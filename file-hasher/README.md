# File Hasher

## Description
A Python utility that calculates the SHA-256 hash of a file. This tool is used to verify file integrity and ensure that files have not been tampered with.

## Features
- Generates secure SHA-256 hashes.
- Fast and lightweight.
- Easy to use via command line.

## How to Run
1. Ensure Python is installed.
2. Run the script: `python3 file_hasher.py`
3. Provide the file path when prompted.
